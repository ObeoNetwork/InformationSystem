Cinematic package

Overview

This documentation describes services and attributes available on the Cinematic concepts available in M2Doc generation templates.

Each concept is described in a page of it's own. Services and attributes are described using

  • a name,
  • the syntax to be used in M2Doc generation templates,
  • a description.

For some attributes and services, a usage example is provided.

Viewpoints

There are three available representations on Cinematic's elements :

  • UI Structure : a tree allowing to structure the Cinematic model contents,
  • Package Diagram : a diagram allowing to represent and edit the package strucure,
  • Flow Diagram : a flow diagram allowing to represent and edit a flow.
