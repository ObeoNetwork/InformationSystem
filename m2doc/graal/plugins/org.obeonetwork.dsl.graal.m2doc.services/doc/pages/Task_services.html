Services and attributes for the Task concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
metadatas List of MetaDatas {m:t.metadatas} access to a container of Meta datas which are key value maps.
description String {m:t.description} description of the task. description : {m:t.description}
keywords List of String {m:t.keywords} List of keywords that can be used to index the task. key words : {m:t.keywords->sep(',')}
behaviours List of Behaviors {m:t.behavior} behaviors where the task is involved
bindingRegistries List of BindingRegistries {m:t.bindingRegistries} binding registries associated to the task
version Integer {m:t.version} Version number of the elmeent.
createdOn Date {m:t.createdOn} Date where the task has been created.
modifiedOn Date {m:t.modifiedOn} Date where the task has been last modified.
name String {m:t.name} name of the task.
nodes List of Nodes {m:t.nodes} nodes of the activity
transitions List of Transitions {m:t.transitions} transitions between the activity's nodes
subActivities List of Loops {m:t.subActivities} sub activities
id String {m:t.id} task technical identifier
uses List of Tasks {m:t.uses} list of tasks that are used by this one.
preconditions String {m:t.preconditions} list of preconditions of the task
postconditions String {m:t.postconditions} list of post conditions of the task
actors List of Actors {m:t.actors} list of actors that may carry the task.

Services

Name Returned type M2Doc Syntaxe Description Example
allChildrenAndTask List of EObjects {m:t.allChildrenAndTask()} returns a list containing a Task and all its children.
annotation String {m:v.annotation('SIZE')} returns the body of the annotation attached to the specified object with the specified title.
getDocumentationLinks List of Annotation objects {m:v.getDocumentationLinks()} returns all annotations with type 'DocumentationLink' attached to the specified object.
