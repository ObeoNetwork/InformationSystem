Services and attributes for the DomainClass concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
metadatas List of MetaData {m:v.metadatas} access to a container of Meta datas which are key value maps.
description String {m:v.description} description of the operator. description : {m:v.description}
keywords List of String {m:v.keywords} keywords used to index the element. keywords : {m:v.keywords->sep(',')}
behaviours List of Behaviour {m:v.behaviors} behaviors related to the element.
bindingRegistries List of BindingRegistries {m:v.bindingRegistries} binding registries related to the element.
version EInt {m:v.version} version number.
createdOn Date {m:v.createdOn} creation date.
modifiedOn Date {m:v.modifiedOn} modification date.
name String {m:v.name} name of the element.
typeDefinition TypeDefinition {m:v.typeDefinition} type definitions this Domainclass is in.
supertype DomainClass {m:v.supertype} DomainClass this DomainClass inherits from.
associatedTypes List of DomainClass {m:v.associatedTypes} list of associated DomainClasses.
ownedAttributes List of Attributes {m:v.ownedAttributes} list of owned attributes.
attributes List of Attributes {m:v.attributes} list of attributes of the DomainClass (including attributes from the super DomainClass)
ownedReferences List of References {m:v.ownedReferences} list of owned references
references List of References {m:v.references} list of References to other DomainClass including inherited ones.
properties List of Properties {m:v.properties} list of properties defined in the DomainClass.
estimatedVolumetry Integer
historized Boolean {m:v.historized} true when the element is historized.

Services

Name Returned type M2Doc Syntaxe Description Example
isHistorized String {m:v.isHistorizedf()} returns the string 'Oui' when true and the string 'Non' when false
checkHistorized String {m:v.checkHistorized()} returns the string 'X' when true and the empty string when false
fullQualifiedName String {m:v.fullQualifiedName()} returns the fully qualified name of the domain class. That is the name of the class prefixed by it's containing packages.
annotation String {m:v.annotation('SIZE')} returns the body of the annotation attached to the domain class with the specified title.
