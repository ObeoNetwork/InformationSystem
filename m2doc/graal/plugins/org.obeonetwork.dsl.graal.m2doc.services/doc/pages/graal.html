Graal package

Overview

This documentation describes services and attributes available on Graal models in M2Doc generation templates.

Each concept (System, Task, User action, etc.) is described in a page of it's own. Services and attributes are described using

  • a name,
  • the syntax to be used in M2Doc generation templates,
  • a description of the attribute or service

For some attributes and services, a usage example is provided.
