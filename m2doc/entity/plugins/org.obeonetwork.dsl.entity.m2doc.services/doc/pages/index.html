Entity package

Overview

This documentation describes services and attributes available on the Entity concepts available in M2Doc generation templates.

Each concept is described in a page of it's own. Services and attributes are described using

  • a name,
  • the syntax to be used in M2Doc generation templates,
  • a description.

For some attributes and services, a usage example is provided.

Viewpoints

Entity Views

The Entity Views viewpoint provides three representations :

  • Entities Diagram : a diagram allowing to represent and edit the Entities of a Namespace,
  • Entities Namespaces Hierarchy : a diagram allowing to represent and edit the Namespace hierarchy of an Entity model,
  • Entity Table : a table showing the Entities of an Entity model with their number of attributes, estimated volumetry and historized setting.
