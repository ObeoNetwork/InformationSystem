Entity package

Overview

This documentation describes services and attributes available on the Entity concepts available in M2Doc generation templates.

Each concept is described in a page of it's own. Services and attributes are described using

  • a name,
  • the syntax to be used in M2Doc generation templates,
  • a description.

For some attributes and services, a usage example is provided.

Documentation of M2Doc Services for the Entity package
