Services and attributes for the Constraint concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:cstr.ID} returns the identifier of the constraint
comments String {m:cstr.comments} returns the comments stored on the constraint
techID String {m:cstr.techID} returns the technical id of the constraint
name String {m:cstr.name} returns the name of the constraint
expression String {m:cstr.expression} returns the expression of the constraint
owner Table {m:cstr.table} returns the table owing the constraint
