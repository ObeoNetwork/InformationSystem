Services and attributes for the Column concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:col.ID} returns the identifier of the column column's identifier: {m:col.ID}
comments String {m:col.comments} returns the comments stored on the columns
techID String {m:col.techID} returns the technical id of the column
name String {m:col.name} returns the name of the column
nullable True/False {m:col.nullable} returns true if the column may contain null value and false otherwise
defaultValue String {m:col.defaultValue} returns the column's default value
indexes List of Indexes {m:col.indexes} returns the list of indexes used by the column the list of indexes of the column : {m:for idx | col.indexes}
- {m:idx.name}
{m:endfor}
indexElements List of index elements {m:col.indexElements} returns the list of index elements associated to the column
primaryKey Primary Key {m:col.primaryKey} returns the primary key associated to the column if it's in a primary key
foreignKeys Foreign Key {m:col.foreignKeys} returns the foreign keys this column is in
foreignKeyElements List of foreign key elements {m:col.foreighKeyElements} returns the foreign key elements for that column
sequence Sequence {m:col.sequence} return the sequence used by this column
owner Table {m:col.owner} return the table owning this column
autoincrement True/False {m:col.autoincrement} returns true when the column is automatically incremented and false otherwise
inPrimaryKey True/False {m:col.inPrimaryKey} returns true when the column is in a primary key, false otherwise
inForeignKey True/False {m:col.inForeignKey} returns true when the column is in a foreign key and false otherwise
unique True/False {m:col.unique} returns true when the column may not have duplicate values and false otherwise

isInPrimaryKey String {m:col.isInPrimaryKey()} returns the string 'Oui' when the column is in a primary key and the string 'Non' otherwise
checkInPrimaryKey String {m:col.checkInPrimaryKey()} returns the string 'X' when the column is in a primary key and the empty string otherwise
isInForeignKey String {m:col.isInForeignKey()} returns the string 'Oui' when the column is in a foreign key and the string 'Non' otherwise
checkInForeignKey String {m:col.checkInForeignKey()} returns the string 'X' when the column is in a foreign key and the empty string otherwise
isUnic String {m:col.isUnic()} returns the string 'Oui' when when the column may not have duplicate values and the string 'Non' otherwise
checkUnique String {m:col.checkUnique()} returns the string 'X' when when the column may not have duplicate values and the empty string otherwise
typeName String {m:col.typeName()} Returns the name of the column's type. The column has type {m:col.typeName()}
typeLength String {m:col.typeLength()} Returns the length of the column's type:
  • if the type has a length, the result is it's length
  • if the type has a length and a precision, the result is the length and the precision separated by a comma (e.g. "255,8").
  • returns the empty string in any other case
 The column has type {m:col.typeName()}

checkMandatory String {m:col.checkMandatory()} returns the string 'X' when when the column is mandatory (not nullable)
defaultValue String {m:defaultValue()} Returns the default value associated to a column. If there's no default value, returns the empty string. This column's default value is {m:col.defaultValue()}
