Services and attributes for the ForeignKey concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:fk.ID} returns the identifier of the foreign key The primary key identifier is {m:pk.ID}
comments String {m:fk.comments} returns the comments stored on the foreign key
techID String {m:fk.techID} returns the technical id of the foreign key
name String {m:fk.name} returns the name of the foreign key
elements List of foreign key elements {m:fk.elements} returns the list of foreign key elements associated to this FK The list of foreign key elements of the foreign key is
{m:for elt | fk.columns}
- {m:elt.name}
{m:endfor}
owner Table {m:fk.owner} returns the table owning this FK
target Table {m:fk.target} returns the target of the FK
