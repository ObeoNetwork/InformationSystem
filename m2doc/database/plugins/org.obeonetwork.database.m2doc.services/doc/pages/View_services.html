Services and attributes for the View concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:view.ID} returns the identifier of the View The view's identifier is {m:idx.ID}.
comments String {m:view.comments} returns the comments stored on the view
techID String {m:view.techID} returns the technical id of the view
name String {m:view.name} returns the name of the view
owner database or schema {m:view.owner} returns the database or the schema owning the view
query String {m:view.query} returns the query that defines the view
columns List of view elements {m:view.columns} returns the list of columns in the view The list of columns of the view is
{m:for col | view.columns}
- {m:col.name}
{m:endfor}
tables List of view elements {m:view.tables} returns the list of tables in the view The list of tables of the view is
{m:for table | view.tabless}
- {m:table.name}
{m:endfor}
