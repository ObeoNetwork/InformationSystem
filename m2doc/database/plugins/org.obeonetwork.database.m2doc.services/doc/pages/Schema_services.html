Services and attributes for the Schema concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:schm.ID} returns the identifier of the schema The schema element identifier is {m:sch.ID}
comments String {m.schm.comments} returns the comments stored on the schema
techID String {m.schm.techID} returns the technical id of the schema
name String {m.schm.name} returns the name of the schema
tables List of tables and views {m.schm.tables} returns the list of tables and views of the schema The list of tables of the schema is
{m:for table | sch.tables}
-{m:table.name}
{m:endfor}
sequences List of sequences {m.schm.sequences} returns the list of sequences of the schema

Services

Name Returned type M2Doc Syntaxe Description Example
tables List of tables {m:schema.tables()} returns the set of tables directly stored in the schema the list of schema tables : {m:for table | schema.tables()}table {m:table.name},{m:endfor}
views List of views {m:schema.views()} returns the set of views directly stored in the schema. the list of database views : {m:for table | schema.views()}view {m:view.name},{m:endfor}
