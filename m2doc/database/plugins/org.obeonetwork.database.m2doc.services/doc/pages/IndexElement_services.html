Services and attributes for the IndexElement concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:idxe.ID} returns the identifier of the index element The index element identifier is {m:idxElt.ID}
comments String {m:idxe.comments} returns the comments stored on the index element
techID String {m:idxe.techID} returns the technical id of the index element
column Column {m:idxe.column} returns the column associated index
asc True/False {m:idxe.asc} returns True when the index is ascending, False otherwise

Services

Name Returned type M2Doc Syntaxe Description Example
isAscending String {m:idxe.isAscending()} returns the string 'Oui' when the index is ascending and the string 'Non' otherwise
checkAscending String {m:idxe.checkAscending()} returns the string 'X' when the index is ascending and the empty string otherwise
