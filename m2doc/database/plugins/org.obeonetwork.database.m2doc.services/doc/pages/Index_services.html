Services and attributes for the Index concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m:idx.ID} returns the identifier of the Index The index's identifier is {m:idx.ID}.
comments String {m:idx.comments} returns the comments stored on the index
techID String {m:idx.techID} returns the technical id of the index
name String {m:idx.name} returns the name of the index
qualifier String {m:idx.qualifier} returns the qualifier of the index
elements List of index elements {m:idx.elements} returns the index elements associated to this index The index elements associated to this index are
/{m:for elt | idx.elements}
-{m:elt.name}
{m:endfor}
unique True/False {m:idx.unique} returns true when the index may not contain duplicate values and false otherwise
cardinality Integer {m:idx.cardinality} returns the cardinality of the index
indexType String {m:idx.indexType} returns the type of the index
owner Table {m:idx.owner} returns the table owning the index

Services

Name Returned type M2Doc Syntaxe Description Example
isUnique String {m:idx.isUnique()} returns the string 'Oui' when the index may not contain duplicate values and the string 'Non' otherwise
checkUnique String {m:idx.checkUnique()} returns the string 'X' when the index may not contain duplicate values and the empty string otherwise
