Services and attributes for the Sequence concept

Attributes

Name Returned type M2Doc Syntaxe Description Example
ID String {m.seq.ID} returns the identifier of the sequence The sequence identifier is {m:seq.ID}
comments String {m.seq.comments} returns the comments stored on the sequence
techID String {m.seq.techID} returns the technical id of the sequence
name String {m.seq.name} returns the name of the sequence
start Integer {m.seq.start} returns the start value of the sequence.
increment Integer {m.seq.increment} returns the value of the increment for the sequence
minValue Integer {m.seq.minValue} returns the minimum value of the sequence
maxValue Integer {m.seq.maxValue} returns the maximum value of the sequence
cacheSize Integer {m.seq.cacheSize} returns the cache size of the sequence
cycle True/False {m.seq.cycle} returns true when the sequence cycles when reaching the max or min value
columns List of columns {m.seq.columns} List of columns using the sequence. The list of columns using the sequence is
{m:for col | seq.columns}
- {m:col.name}
{m:endfor}

Services

Name Returned type M2Doc Syntaxe Description Example
isCycle String {m.seq.isCycle()} returns the string 'Oui' if the sequence cycles when reaching the max or min value and the string 'Non' otherwise
checkCycle String {m.seq.checkCycle()} returns the string 'X' when the sequence cycles when reaching the max or min value and the empty string otherwise
columns List of columns {m.seq.columns()} returns the list of columns using the sequence. Returns the empty list if the sequence is null. The list of columns using the sequence is
{m:for col | seq.columns()}
- {m:col.name}
{m:endfor}
