SOA package

Overview

This documentation describes services and attributes available on the SOA concepts available in M2Doc generation templates.

Each concept is described in a page of it's own. Services and attributes are described using

  • a name,
  • the syntax to be used in M2Doc generation templates,
  • a description.

For some attributes and services, a usage example is provided.

Viewpoints

SOA Views

The SOA Views viewpoint provides four representations :

  • Component Contract Diagram : a diagram allowing to represent and edit the required and provided Services of a Component,
  • DTO Diagram : a diagram allowing to represent and edit the DTOs of a Namespace,
  • DTO Namespaces Hierarchy : a diagram allowing to represent and edit the Namespace hierarchy of a System,
  • SOA Diagram : a diagram allowing to represent and edit the Components of a System.
